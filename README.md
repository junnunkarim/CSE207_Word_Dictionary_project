# CSE207 Project - Word Dictionary
A small project about implementing a word dictionary with binary search tree using C++.

- University: East West University
- Course: CSE207 Data Structure

# Directory Structure
```
├── CMakeLists.txt
├── README.md
├── data
│   └── database
├── docs
├── include
│   ├── bst.h
│   ├── bst.tpp
│   ├── list.h
│   ├── list.tpp
│   ├── utility.h
│   └── word.h
├── outline.md
├── src
│   ├── main.cpp
│   ├── utility.cpp
│   └── word.cpp
└── tests
    ├── test_bst.cpp
    ├── test_list.cpp
    └── test_word.cpp
```
